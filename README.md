# 🛒 Django E-commerce Shopping Website

A full-stack E-commerce web application developed using Python and Django. This project allows users to browse products, search items, filter products by category, manage a shopping cart, and place orders through a simple and user-friendly interface.

---

## 🚀 Features

- User Registration & Login
- Session-Based Authentication
- Product Listing
- Product Details Page
- Category-wise Product Filter
- Product Search
- Add to Cart
- Increase / Decrease Cart Quantity
- Remove Cart Items
- Checkout System
- Order Management
- Django Admin Panel

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/RAHUL-84/django-ecommerce-website.git
```

Go to the project directory:

```bash
cd django-ecommerce-website
```

Install Django:

```bash
pip install django
```

Apply migrations:

```bash
python manage.py makemigrations
python manage.py migrate
```

Run the server:

```bash
python manage.py runserver
```

Open in your browser:

```
http://127.0.0.1:8000/
```
