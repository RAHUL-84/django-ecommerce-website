🟢 1. Category Model Kaise Banayi?
🔹 Step 1

📁 File Banayi

store/models/category.py
from django.db import models

class Category(models.Model):
    name = models.CharField(max_length=100)

⬇

🔹 Step 2

📁 models/init.py

from .category import Category

⬇

🔹 Step 3

📁 admin.py

admin.site.register(Category)

⬇

🔹 Step 4

🖥 Migration Banayi

python manage.py makemigrations

⬇

🔹 Step 5

🖥 Database Me Table Banayi

python manage.py migrate

⬇

✅ Result

✔ Category Table Database me ban gayi.

✔ Django Admin me Add aur Change option aa gaya.

⭐ Interview Me Ek Line Me

🟢 Answer:

Sabse pehle maine Model Class banayi. Uske baad models/__init__.py me import kiya. Fir admin.py me admin.site.register() se register kiya. Uske baad python manage.py makemigrations chalaya jisse migration file bani. Fir python manage.py migrate chalaya jisse database me table create ho gayi. Iske baad Django Admin me us model ka Add aur Change option automatically dikhne laga.
