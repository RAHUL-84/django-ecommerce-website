# 🟣 PART-6

## 🔷 makemigrations Kyu Chalaya?

### ✅ Answer

Jab bhi Model me Change karte hain

Django ko batana padta hai ki

Database bhi Update karna hai.

Uske liye

```bash
python manage.py makemigrations
```

chalate hain.

Ye Migration File banata hai.

---

## 🔷 migrate Kyu Chalaya?

```bash
python manage.py migrate
```

### ✅ Answer

Migration File ko Database me Apply karta hai.

Uske baad Database me Actual Table Create hoti hai.


likha.

```django

```

ke andar sirf page ka alag content likha.
